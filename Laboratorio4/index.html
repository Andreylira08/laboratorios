Fundamentos de JavaScript

¿Que diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación orientado a objetos. Javascript es un lenguaje que permite programar sin necesidad de crear clases y a diferencia de Java no requiere hacer la programación orientándose a objetos, aunque sí lo permite.

2. setMonth()
3. setDate()
4. setHours()
5. setMinutes()
6. setSeconds()
7. setMilliseconds
8. setTime(),

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

1. Push() – Inserta un elemento al final del arreglo.
2. unshift() – Inserta un elemento al inicio del arreglo.
3. pop() – Remueve un elemento del final del arreglo.
4. shift() – Remueve un elemento del principio del arreglo.
5. slice() – Crea una copia sombra del arreglo.

¿Cómo se declara una variable con alcance local dentro de una función?

En JavaScript, puedes declarar una variable con alcance local dentro de una función utilizando let o const.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Como las variables globales pueden ser modificadas desde cualquier parte del código, una función puede accidentalmente cambiar su valor, causando errores difíciles de depurar pero En programas pequeños, una variable global puede ayudar a mantener un estado simple sin necesidad de estructuras más complejas.
